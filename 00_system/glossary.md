# EmberSense Glossary

## Terms

**MFCC**: Mel-Frequency Cepstral Coefficients - audio feature representation
**DSP**: Digital Signal Processing
**MCU**: Microcontroller Unit
**VOC**: Volatile Organic Compounds
**LUFS**: Loudness Units relative to Full Scale - audio normalization
**SNR**: Signal-to-Noise Ratio
**F1**: Harmonic mean of precision and recall
**ROC**: Receiver Operating Characteristic curve
**Edge Impulse**: Platform for edge ML development

## Acronyms
- **EI**: Edge Impulse
- **CNN**: Convolutional Neural Network
- **GRU**: Gated Recurrent Unit
- **SVM**: Support Vector Machine
- **RF**: Random Forest
- **NASA**: National Aeronautics and Space Administration
- **NOAA**: National Oceanic and Atmospheric Administration

## File Types
- **.eim**: Edge Impulse Model format
- **.flac**: Free Lossless Audio Codec
- **.parquet**: Columnar storage format
- **.npy**: NumPy array format
